Slow Tox Visualized
Client HUD for Slow Tox intoxication and buffs

Built for the Teotale Vintage Story server (www.teotale.com). Shows blood-alcohol level, a color palette tied to how drunk you are, and a strip of active effects - without opening character stats.

Important - Slow Tox is required

Without Slow Tox (mods.vintagestory.at/slowtox), Slow Tox Visualized has no meaningful role: every status and tooltip is tied to the intoxication rules and stats from that mod. Install Slow Tox first - this client HUD only surfaces what Slow Tox already computes.

Main HUD
Mug and big number
Intoxication percent (HUD-style), with fill and stroke colors from your palette.
Gear
SVG icon tinted from the palette, with configurable rotation (stroke and size in JSON).
Status strip

Icons follow Slow Tox logic: damage reduction, HP regen, stability, melee, mining, slow, poison - short tooltips on hover.

Configuration
slowtoxvisualized-hudlayout.json in your game ModConfig folder.
After edits - reload layout (chat notification / hotkey as configured).

Requires the Slow Tox mod for buff stats. Game version - see modinfo.json.
